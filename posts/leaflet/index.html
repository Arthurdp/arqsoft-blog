Este documento descreve parte da arquitetura do projeto Leaflet. Essa descrição foi baseada principalmente no modelo C4.

Descrição Geral sobre o Leaflet

O leaflet é uma biblioteca leve em javascript que permite o uso de mapas de interativos compatíveis com mobile. Permite o uso de plugins para extender as funcionalidades e tem como objetivo oferecer todas as funcionalidades que podem vir a serem necessárias em mapas.
